# Prompt Engineering Log

This log documents any AI-assisted development (e.g., using LLMs for code generation, architectural advice, or debugging).

## Format
For each significant interaction, record:
- **Date:** 
- **Goal/Context:** What were you trying to achieve?
- **Prompts Used:** The exact prompt(s) given to the AI.
- **Output/Result:** A summary of the AI's response.
- **Iterations:** Any refinements or corrections needed.
- **Lessons Learned:** Takeaways or optimizations for future prompts.
